Real-Time Bus Tracker for Washington D.C.

The "WMATA Bus Tracker" is a web application that leverages HTML, CSS, and JavaScript to offer real-time public transport route information. It integrates the Washington Metropolitan Area Transit Authority (WMATA) API and utilizes the Mapbox API to display interactive maps. Users can track buses and view their routes, making it a practical tool for commuters. My project highlights my skills in web development and API integration.

PacMen Exercise

For the "PacMen Exercise," I created a simple interactive web page using HTML, CSS, and JavaScript. This project allowed me to explore array manipulation techniques and DOM interactions. Visitors can add PacMen to the screen with the click of a button and watch them move around the page at random velocities. The PacMen change position to stay within the frame, making it a dynamic and engaging exercise.
